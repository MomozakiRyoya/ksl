# FSL（福岡ポーカーリーグ）— 仕様書

## プロジェクト概要
福岡のポーカーチームリーグ戦プレイヤー向けアプリ。最新情報・試合結果・順位表・選手情報を提供するWebアプリケーション。

## 目的・ゴール
- リーグ参加プレイヤーへの試合結果・最新情報の配信
- 順位表・スタッツのリアルタイム更新
- チーム・選手情報の一元管理

## 対象ユーザー
- FSL（福岡ポーカーリーグ）参加プレイヤー
- リーグ観戦ファン
- チーム運営者

## 技術スタック
- **フロントエンド**: Next.js / TypeScript / Tailwind CSS
- **バックエンド**: Next.js API Routes
- **DB**: Supabase (PostgreSQL)
- **ホスティング**: Vercel

## 主要機能
1. 最新情報・お知らせ配信
2. 試合結果の登録・表示
3. 順位表（個人・チーム）
4. 選手プロフィール・成績
5. スケジュール管理
6. 過去リーグ結果アーカイブ
